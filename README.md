alerta.io
=========

Alerta monitoring system web site.


Installation
------------

    $ sudo gem install jekyll
    $ sudo gem install redcarpet
    $ git clone https://github.com/alerta/alerta.github.io
    $ cd alerta.github.io
    $ jekyll serve -w
