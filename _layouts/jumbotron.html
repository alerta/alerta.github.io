Why Alerta?

Scalable monitoring and alert visualisation. A minimal installation can be deployed quickly and easily extended as monitoring requirements and confidence grow. There are integrations available with Riemann, Nagios, Zabbix, Kibana and Sensu. Integrating bespoke systems is easy using the API or command-line tool.

Alert API

Alerts are submitted in JSON-format to a HTTP API. Alerts can be queried from the command-line or viewed in a responsive web console optimized for desktop, tablet and mobile. User access can be added using Google OAuth2 and programatic access using API keys.
