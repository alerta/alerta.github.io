Release 4.4 is available.

Alerta now supports MongoDB 3.0 as the backend database. Backward compatibility with MongoDB 2.x is still supported. And don't forget about the out-of-the-box integrations with CloudWatch, Pingdom and PagerDuty added in earlier 4.x releases.

Try it now

Keep track of individual alerts.

Logged in users can manage and watch alerts that are of interest to them. So to make user management as simple as possible there is built-in support for OAuth user logins using Google, GitHub and Twitter as well as Basic Auth for situations where OAuth isn't available.

Learn more

Deploy using the Heroku button.

Heroku has introduced a new faster and easier way to deploy apps. Simply click the button in the GitHub repo README for alerta or the web ui to get going in minutes.

Deploy to Heroku

Explore the Alerta API.

Developers can now use a web UI to explore some of the most important features of the Alerta API -- sending and querying for alerts. Any alerts sent to the API using the explorer can be viewed in the explorer or the web console available online.
